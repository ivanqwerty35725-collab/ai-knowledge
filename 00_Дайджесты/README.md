# 🗓️ Дайджесты

Здесь лежат ежедневные сводки новостей. Файл на каждый день: `ГГГГ-ММ-ДД.md`.

## Что в дайджесте
- **5–7 главных новостей** за последние сутки, сгруппированы по разделам:
  - 🔬 Технологии и наука
  - 🧩 Системный анализ
  - 🏠 Повседневная жизнь
- Каждая новость: 2–3 предложения простыми словами + строка «зачем вам» + ссылка «Подробнее».
- Чтение ~10 минут.

## Ритм
- **Ежедневно (утро)** — обычный дайджест.
- **Пятница** — `ГГГГ-ММ-ДД_неделя.md`: 3–5 главных тем недели + «что попробовать».
- **Раз в месяц** — `ГГГГ-ММ_итоги.md`: обзор и ревизия карточек ⭐.
