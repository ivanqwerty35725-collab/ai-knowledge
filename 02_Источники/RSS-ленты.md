# 📡 RSS-ленты

«Лента» — это встроенный в сайт список свежих публикаций (с датами), сделанный для программ.
Эти ленты использует **бот-дайджест**, и их же можно открыть в любой читалке (Feedly, Inoreader):
импортируй файл `RSS-ленты.opml` из этой папки.

> Статус: ✅ — лента проверена «вживую» (июнь 2026); ◻️ — стандартный адрес, обычно стабилен (бот пропустит, если не отвечает).
