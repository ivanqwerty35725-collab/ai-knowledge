# 🏠 Дашборд

Главная страница базы. Списки ниже **собираются сами** из «анкет» карточек.

> ⚙️ Чтобы заработало, нужен бесплатный плагин **Dataview** (как установить — в [[README]], раздел «Ваши шаги»). Пока плагина нет, вы увидите серые блоки с кодом — это нормально, после установки они превратятся в живые таблицы.

---

## ⭐ Хочу попробовать
*Карточки, помеченные статусом «применить».*

```dataview
TABLE WITHOUT ID file.link AS "Карточка", раздел AS "Раздел", источник AS "Источник"
FROM "01_Карточки"
WHERE статус = "применить"
SORT дата DESC
```

## 🆕 Появилось за последние 7 дней

```dataview
TABLE WITHOUT ID file.link AS "Карточка", раздел AS "Раздел", статус AS "Статус"
FROM "01_Карточки"
WHERE дата >= date(today) - dur(7 days)
SORT дата DESC
```

## 📥 Входящие — разобрать
*Сохранённое наспех, что ждёт оформления.*

```dataview
TABLE WITHOUT ID file.link AS "Карточка", источник AS "Источник", дата AS "Дата"
FROM "01_Карточки"
WHERE статус = "входящие"
SORT дата DESC
```

## 📊 Сколько карточек по разделам

```dataview
TABLE WITHOUT ID length(rows) AS "Карточек"
FROM "01_Карточки"
WHERE статус
GROUP BY раздел AS "Раздел"
```
