# 🎴 Флешкарты — как пользоваться

Чтобы знания не выветривались, превращаем их в карточки «вопрос → ответ».
Программа сама показывает чаще то, что вы забываете, и реже то, что уже знаете
(это называют «интервальное повторение» — приём как в Duolingo).

## Что нужно один раз
Бесплатный плагин **Spaced Repetition** для Obsidian (установка — в [[README]], раздел «Ваши шаги»).

## Как повторять
1. Нажми иконку флешкарт на левой панели Obsidian (или команда «Spaced Repetition: Review flashcards»).
2. Прочитай вопрос, вспомни ответ, нажми «показать».
3. Оцени себя: **Hard** (трудно) → карточка вернётся скоро; **Good/Easy** → вернётся нескоро.
4. 3–5 минут в день достаточно.

## Как устроена карточка
Плагин ищет карточки в заметках, где есть тег `#flashcards`. Формат — одна строка:
```
Вопрос::Ответ
```
Есть и двусторонние (спрашивает в обе стороны):
```
Термин:::Определение
```

## Где брать карточки
- [[Флешкарты/Глоссарий — карточки|Глоссарий — карточки]] — термины ИИ (уже готовы).
- Добавляй свои: новый термин или приём из дайджеста → строкой `Вопрос::Ответ` в любой файл с тегом `#flashcards`.
