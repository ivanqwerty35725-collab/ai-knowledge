# 🧠 База знаний по ИИ

Личная база для ежедневного чтения кратких сводок об искусственном интеллекте — простым языком и с упором на практическую пользу. Источники русские и английские, всё излагается на понятном русском.

> Лучше всего открывать в **Obsidian** (бесплатная программа для заметок): тогда работают ссылки `[[...]]`, [[Дашборд]] с автосписками и флешкарты. Это ваши же файлы — Obsidian ничего не ломает.

## 🗓️ Ежедневный дайджест
Собирается **автоматически в облаке каждое утро (~08:00 МСК)** из RSS-лент. Приходит на выбранный канал (почта/Telegram/Notion) и/или в репозиторий, а в папку `00_Дайджесты` сохраняется при синхронизации.
Управление расписанием: https://claude.ai/code/routines

## 📇 Как сохранять новости
Понравилось в дайджесте — заведи карточку по [[_Шаблон_карточки|шаблону]] в папке `01_Карточки`: вверху «анкета» (дата, раздел, статус, теги), ниже текст. Раздел задаёт **поле**, не папка. Из «анкет» сам собирается [[Дашборд]].

## ⚙️ Настройка
1. ✅ **Obsidian установлен**, хранилище зарегистрировано.
2. ✅ **Плагины Dataview и Spaced Repetition** предустановлены. Если не активны — Настройки → Community plugins → выключить «Restricted mode».
3. ⏳ **GitHub** — приватный репозиторий: бот складывает туда выпуски (не теряются, не повторяются), на компьютер скачиваются.
4. ⏳ **Канал доставки** — почта/Telegram/Notion на claude.ai/customize/connectors.

> По шагам 3–4 просто скажите — проведу по пунктам.

## ▶️ С чего начать каждый день
Открой свежий файл в `00_Дайджесты` → заинтересовало, перейди по ссылке «Подробнее» → стоящее сохрани карточкой (статус `применить`) → незнакомый термин смотри в [[Глоссарий]].
