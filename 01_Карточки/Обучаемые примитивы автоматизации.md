# Обучаемые примитивы автоматизации: покажи один раз → получи навык

**Простыми словами:**
Инструменты для ИИ-агентов учатся повторять ваши действия: показываете рабочий процесс один раз — система превращает его в многоразовый «навык». А саму автоматизацию всё чаще задают обычными словами.

**Зачем мне:**
Прямо применимо в работе: повторяющиеся задачи (сбор требований, отчёты, проверки) можно превратить в переиспользуемые автоматизации.

**Подробнее (для углублённого изучения):**
- **@OpenAIDevs — Codex Record & Replay:** один раз продемонстрировать рабочий процесс → превратить его в *проверяемый* (verifiable) навык.
- **@cursor_ai — /automate:** Cursor сам настраивает триггеры/инструкции/инструменты по задаче на естественном языке; триггеры с эмодзи в Slack, триггеры в GitHub, «computer use» для облачных агентов.
- **@ClaudeDevs — Artifacts в Claude Code:** агент превращает текущую работу в интерактивные страницы, которыми можно делиться. **@_catwu**: это уже изменило внутренние процессы — обсуждение изменений в архитектуре и обмен прототипами.
